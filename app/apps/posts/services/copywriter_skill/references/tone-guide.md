# Guia de Tom por Segmento de Mercado

Referência rápida para calibrar o tom sem depender apenas do briefing.
Se o cliente não especificou o tom, use o segmento para inferir o mais adequado.

## Regras de Adaptação de Tom

### Se o briefing diz um tom mas o segmento sugere outro:
Priorize o briefing — o cliente conhece sua marca. Mas registre no campo `notes` da variante
se houver tensão entre tom e segmento.

### Tons que nunca funcionam juntos:
- `urgente` + `inspiracional` na mesma frase (escolha um por parágrafo)
- `humorístico` + `profissional` no mesmo copy (exceto LinkedIn com tom leve)
- `educativo` no CTA (CTA é sempre ação — nunca explicação)

### Emojis por tom:
| Tom | Uso de emoji |
|---|---|
| descontraído | Liberado — até 1 por parágrafo |
| urgente | Apenas ⚡ ⏳ 🔥 — com moderação |
| inspiracional | 🌟 ✨ apenas no início ou fim |
| profissional | Nenhum (exceto LinkedIn com público jovem) |
| educativo | 👉 — para listas — nada decorativo |
| humorístico | Liberado com cautela — emoji não substitui piada |
